So when we look at our prior code, we defines a variable called ‘df$name.’ What we want to find out is what type variable it is.

df<-Teams%>%
  filter(yearID==1980)%>%
  select(name,HR)%>%
  arrange(HR)
df$name<-factor(df$name,levels=df$name)

if we type “Teams” into the r studio command line it retuens the information in the table “teams.” if we trple class(Teams) it returns the “type” [1] “data.frame” telling us it is a data frame.

if we type “class(df$name)” into r studio is retuns [1} “factor” as the variable type

factors have levels. again if we type “factor(df$name)” into the Rstudio command line it returns the names of the 25 teams & ends with “26 Levels: New York Mets San Diego Padres … Milwaukee Brewers” Telling us tehre are 26 levels in the factor.

What all this tels us is that we can modify & control the types of variables.
