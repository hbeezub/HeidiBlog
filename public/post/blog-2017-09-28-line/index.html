Sep 28, 2017 Data650, Data Visualization, Rstudio, ggplot, Line Plot, Google Anniversary

A Line graph is great way to show data over time.

Ok we need to go through all the circus of installing packages and bringing them into R studio with library:

library(devtools)
library(Lahman)
library(dplyr)
library(ggplot2)
library(ggiraph)

We’ll extract some dat afor our time series:

df<-Batting%>%
  filter(playerID=='ruthba01')%>%
  select(yearID,HR)

and add a simple line graph

ggplot()+
  geom_line(data=df, aes(x=yearID, y=HR))

Not too exciting: no title, it would be nice to have some points to know numbers. So lets add to our plotting code:

ggplot()+
  geom_line(data=df, aes(x=yearID, y=HR))+
  geom_point(data=df,aes(x=yearID,y=HR))+
  ggtitle("Babe Ruth HomeRuns")

Now that is definately improved. But wait… there’s more. Act now and we can make this an interactive graph with hover points.

-as a special bonus you get the following included at no additional charge.

The funFact for Today, 9-27-2017 is that Google is 19 years old today. Happy Birthday Google. But even of more note is that yesterday was my 19th wedding anniversary. So my marraige is one day older than Google.

g<-ggplot()+
  geom_line(data=df, aes(x=yearID, y=HR))+
  geom_point_interactive(data=df,aes(x=yearID,y=HR,tooltip=HR,data_id=yearID))+
  ggtitle("Babe Ruth HomeRuns")
                       
ggiraph(code=print(g),hover_css = "fill:white;stroke:blue")

It’s outta here…
